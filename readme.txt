Librarys are required for building flowspring:
	*  JRE System Library with jre7 or jdk7
	*  vlcj-2.1.0.jar is required along with jna-3.4.0.jar and platform-3.4.0.jar
				(https://github.com/caprica/vlcj)
	*  For vlcj libvlc and libvlccore is required along with vlcs plugin folder for
	   the selected system and has to be in "./lib/win32", "./lib/win64", "./lib/lin32"
	   or "./lib/lin64"
	*  hsqldb is required (http://hsqldb.org/)
	*  jintellitype-1.3.6 is required (http://code.google.com/p/jintellitype/)
	*  jxgrabkey is required (https://sourceforge.net/projects/jxgrabkey/)
	*  entagged is required (http://entagged.sourceforge.net/)
